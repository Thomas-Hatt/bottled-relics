# Bottled Relics

Bottled Relics is a mod for Slay the Spire that adds 10 relics as potions that last for the combat.

https://steamcommunity.com/sharedfiles/filedetails/?id=3666446085

---
Bottled Potions:
1. Letter Opener (COMMON)
2. Ornamental Fan (COMMON)
3. Strike Dummy (COMMON)
4. Dead Branch (UNCOMMON)
5. Gremlin Horn (UNCOMMON)
6. Kunai (UNCOMMON)
7. Shuriken (UNCOMMON)
8. Warped Tongs (UNCOMMON)
9. Pocket Watch (RARE)
10. Snecko Eye (RARE)
